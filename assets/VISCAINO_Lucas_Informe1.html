usd_tot <- sum(turistas_raw$gasto_total)
turistas_adj <- turistas_raw|> mutate (
  estudio = ifelse(is.na(estudio),"Sin Datos",estudio),
  id_nivel_estudio = ifelse(is.na(id_nivel_estudio),0,id_nivel_estudio)
  )|>
  group_by(id_nivel_estudio,estudio)|>
  summarise(sum_gasto = sum(gasto_total))|> mutate(
    gasto_perc = sum_gasto/usd_tot
    )
turistas_adj <- arrange(turistas_adj,desc(gasto_perc))

turistas_adj$perc_acc <- cumsum(turistas_adj$gasto_perc)

#primer grafico - donut chart
# Create donut data.
donut_data <- turistas_adj

# Compute percentages
donut_data$fraction <- turistas_adj$gasto_perc

# Compute the cumulative percentages (top of each rectangle)
donut_data$ymax <- turistas_adj$perc_acc

# Compute the bottom of each rectangle
donut_data$ymin <- c(0, head(donut_data$ymax, n=-1))

# Compute label position
donut_data$labelPosition <- (donut_data$ymax + donut_data$ymin) / 2

# Label the sections
donut_data$label <- paste0("Valor: \n ", scales::percent(donut_data$gasto_perc))

# Make the plot

Relaciones entre variables
De la figura 3 podemos observar que los destinos m√°s visitados son Argentina y Brasil, con Argentina teniendo mas de 6000 encuestados cuyo motivo de viaje fueron Ocio, Recreo, Vacaciones. Brasil presenta tendencias similares, con una cifra cercana a los 4000 encuestados para la misma categor√≠a. El resto de los Destinos de viaje no muestran una marcada relaci√≥n con su Motivo, pero puede que la resoluci√≥n de la escala de colores de la visualizaci√≥n nos impida apreciar m√°s detalles. En cuanto a la figura 4, se filtraron los datos para analizar solamente Argentina y Brasil como destinos dado lo analizado a partir de la figura 3, y se destacan ciertos patrones.
En la figura 4a se encuentran los per√≠odos hasta el 2020 y se puede apreciar como Argentina ten√≠a una predominancia de Visita familiares/amigos como motivo, mientras que en Brasil predominan los viajes por Ocio, Recreo, Vacaciones. Las tendencias en estas figuras no son demasiado marcadas para Argentina. Sin embargo, Brasil muestra estacionalidad en viajes de Ocio, Recreo, Vacaciones durante los meses de verano (12, 1 y 2).
Hay un periodo no graficado (desde 2020-04 hasta 2021-10) que nos indica el tiempo de aislamiento durante la pandemia por Covid-19 (para Brasil, dicho periodo se extendi√≥ hasta el 2021-11, con los primeros datos obtenidos a partir de 2021-12). Luego de este evento, las proporciones para Argentina cambiaron, siendo mayoritario a partir de 2022-03 los viajes por Ocio,Recreo y Vacaciones, con tendencias inversas a las de Brasil, teniendo picos durante el invierno (a notar el 2022-07 y 2023-06). Asimismo, se detecta un pico interesante en Argentina para 2021-10 y 2021-11 en el que los viajes son puramente por Visita familiares/amigos, mientras que esos periodos en Brasil directamente no registran datos. Esto debe estar asociado a la apertura del aislamiento post-pandemia, en el que los Uruguayos aprovecharon para tener reencuentros luego de los meses de separaci√≥n.
Finalmente, es interesante notar que a partir del 2022-02, la proporci√≥n de viajes con motivo de Compras aumenta para Argentina, lo que puede indicar una ventaja cambiar√≠a para los Uruguayos que eligen Argentina como destino.
Concluyendo, efectivamente hay una vinculaci√≥n entre los Destinos y el Motivo del viaje y, particularmente para las porciones del dataset analizado, se puede ver que hay una estacionalidad en ciertos motivos de viaje, principalmente Ocio, Recreo, Vacaciones para Brasil durante los meses de verano.
